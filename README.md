# SiteNPEK
Расписание. Используется в личных и групповых целях.
Копирование не запрещается, но не наглейте.

Этот сайт — продвинутое расписание для студентов НПЭК.

Расписание (админ): управление расписанием с поддержкой числителя/знаменателя, замен на один день и разделения групп (через разделитель //).
Заметки: создание личных задач и просмотр глобальных заметок.
Глобальные заметки (админ): публикация объявлений и важной информации для всей группы.
Погода и Таймер: автоматический расчет времени до конца пары/перемены и отображение погоды в Новосибирске.
Под капотом: скрытая админ-панель (доступ по паролю) для полного контроля и редактирования контента «на лету».
